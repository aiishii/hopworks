氏姓 的臣
始祖 彦太忍信命三世孫葛城襲津彦
（孝元天皇の後裔）
種別 皇別
凡例 / Category:氏

的氏（いくはうじ）は、「的」を氏の名とする氏族。

概要[編集]

『古事記』によると、玉江氏・生江氏・阿芸那氏とともに葛城長江曾都毘古（かずらき の ながえ の そつびこ）の子孫ということになっている[1]。『新撰姓氏録』山城皇別には、石川朝臣（蘇我氏）と同祖で、彦太忍信命の三世の孫、葛城襲津彦の子孫とある。

的氏はその名前の由来からして弓矢と関係が深い。『日本書紀』巻第十によると、応神天皇の時代に的戸田宿禰（いくは の とだ の すくね）が平群木菟（へぐり の つく)とともに加羅へ派遣されている。そして、葛城襲津彦が帰国しないのは新羅が妨げているからだとして精兵で新羅を攻め、無事弓月の民と襲津彦を奪還したとある[2]。

また巻第十一では、仁徳天皇の時代に、高句麗から献上された鉄の盾と的[3]を、群臣及び百寮が射たが、誰も射通すことができず、ただ一人盾人宿禰（たたひと の すくね）のみが的を射通したという。翌日、盾人宿禰は的戸田宿禰の名を賜り[4]、彼が的氏いくは うじの始祖とされている。戸田宿禰は、5年後に新羅への問責使として朝鮮半島へ派遣されている[5]。別の箇所では、的臣いくは の おみの祖の口持（くちもち）が、仁徳天皇の命で皇后磐之姫命を連れ戻そうとして筒城宮へ向かったが、皇后からの返事を貰えず、悪天候の中ずっと立ち尽くしていたが、そんな兄の姿を見て、妹で侍女の国依媛（くによりひめ）が涙を流して和歌を詠んだ、というエピソードも語られている[6]。

6世紀には、的一族の半島でのさらなる活躍が見られる。『書紀』巻第十九には、推定544年の欽明天皇の任那復興計画に関連して、百済本記の引用文には、任那日本府の官人らしき人物として「烏胡跛臣（うごはのおみ）を遣召（よ）ぶといふ。蓋し是的臣なり」とあり、さらに「今（いま）的臣、吉備臣、河内直等、咸（みな）、移那斯（えなし）・麻都（まつ）が指撝（さしまね）くに従へらくのみ」・「的臣等、等とは吉備弟君臣、河内直等を謂ふなり・新羅に往来（かよ）ひしことは朕（わ）が心に非ず」とある[7]この「的臣」は欽明天皇14年（推定553年）までにはなくなっていることも『書紀』には記載されている[8]。

このほか、用明天皇2年（587年）には、炊屋姫を奉じた蘇我馬子の命により、的真嚙（いくはのまくい）は、佐伯丹経手・土師磐村らとともに、穴穂部皇子・宅部皇子を殺害している[9]。

宮城十二門の郁芳門は、的氏の名に由来し、大炊寮の正面に位置しており、大炊御門とも呼ばれている。淡路国津名郡育波郷（いくはのさと）の名が存在する理由として、淡路島が狩猟地であり、そのことは、

  1. 『書紀』巻第十の応神天皇が実家の吉備へ里帰りさせたの兄媛の様子を見に行く記事[10]
  2. 『書紀』巻第十二の履中天皇の河内飼部（こうちのうまかいべ）の黥（めさきのきず）にまつわる記事や、皇后の薨去の記事[11]
  3. 『書紀』の允恭天皇が赤石の海で真珠（しらたま）を手に入れようとして、白水郎（あま）を潜らせようとした記事[12]

などに見られる。

また、淡路国が御食国であることとも大いに関係があると思われる。津名郡には、阿曇氏や海人、山直氏などが居住しており、的臣も、山の幸や海の幸を育波郷から確保したようである。尾張国海部郡に、「伊久波神社」が存在することも海産物確保のためとみられる。

上記のことから、的氏は軍事的性格とともに、大王の食膳に奉仕する役割も有していたと考えられる[13]。

684年（天武天皇13年）の八色の姓[14]では、的氏は新しい姓を与えられていない。
