印南郡

印南郡（いんなみぐん）は、兵庫県（播磨国）にあった郡。なお、「南」は当て字で方角とは関係ない。

郡域[編集]

1879年（明治12年）に行政区画として発足した当時の郡域は、下記の区域にあたる。基本的に加古川の河口左岸が加古郡、右岸が印南郡である。

  • 高砂市の大部分（高砂町各町・荒井町各町は除く）
  • 姫路市の一部（大塩町各町・的形町各町・別所町各町・飾東町唐端新・飾東町志吹・飾東町北野・飾東町大釜・飾東町大釜新・飾東町清住）
  • 加古川市の一部（加古川以北および加古川町本町・加古川町木村・加古川町西河原・加古川町友沢・加古川町稲屋・上荘町国包）

  • 慶応4年
    • 2月2日（1868年2月24日） - 幕府領が兵庫裁判所の管轄となる。
    • 5月23日（1868年7月12日） - 兵庫裁判所の管轄地域が兵庫県の管轄となる。
    • 6月20日（1868年8月8日） - 交代寄合池田氏が立藩して福本藩となる。
  • 明治3年
    • 一橋徳川家領が兵庫県の管轄となる。
    • 11月23日（1871年1月13日） - 福本藩が廃藩。管轄地域が鳥取藩の管轄となる。
  • 明治4年
    • 7月14日（1871年8月29日） - 廃藩置県により、藩領が姫路県、鳥取県の管轄となる。
    • 11月2日（1871年12月13日） - 第1次府県統合により、全域が姫路県の管轄となる。
    • 11月9日（1871年12月20日） - 飾磨県の管轄となる。
  • 明治9年（1876年）（2町108村）
    • 8月21日 - 第2次府県統合により兵庫県の管轄となる。
    • 福居村・福居新村が合併して別所村となる。
    • 西阿弥陀村・東阿弥陀村が合併して阿弥陀村となる。
    • 柏尾村・吉広村が合併して広尾村となる。
    • 助永村・比室村が合併して永室村となる。
    • 高畑村（現・加古川市米田町平津）が平津村に合併（時期は推定）。
    • 魚崎村が伊保崎村に、魚崎新村が梅井村にそれぞれ改称。
  • 明治10年（1877年）（2町103村）
    • 加古川村の一部が分立して西河原村となる。
    • 中村・芝村が合併して養老村となる。
    • 宮前新村が宮前村に、成井新村が成井村に、原新村が原村に、西飯坂新村が西飯坂村に、奥新村が西牧村にそれぞれ合併。
  • 明治11年（1877年）（1町97村）
    • 下富木村・長慶村・清水村が合併して鼎村となる。
    • 志方西町・志方東町が合併して志方町となる。
    • 寺谷新村・陰山新村が合併して磐村となる。
    • 六本松村・井ノ口新村が井ノ口村（現・加古川市東神吉町西井ノ口）に合併。
    • 2ヶ所存在した原村が、それぞれ上原村（現・加古川市平荘町上原）、下原村（現・加古川市志方町原）に改称。
  • 明治12年（1879年）
    • 1月8日 - 郡区町村編制法の兵庫県での施行により、行政区画としての印南郡が発足。郡役所が曽根村に設置。
    • 加古川村が加古川町に、井ノ口村（現・加古川市東神吉町西井ノ口）が西井ノ口村にそれぞれ改称。（2町96村）

町村制以降の沿革[編集]

  • 明治22年（1889年）4月1日 - 町村制の施行により、以下の町村が発足。特記以外は現・加古川市。（15村）
    • 曽根村（単独村制。現・高砂市）
    • 伊保村 ← 伊保崎村、梅井村、今市村、中島村、中筋村（現・高砂市）
    • 米田村 ← 米田村、米田新村、古新村、塩市村、島村、神爪村（現・高砂市）、平津村、船頭村（現・加古川市）
    • 東神吉村 ← 神吉村、天下原村、升田村、升田新村、砂部村、西井ノ口村
    • 平荘村 ← 山角村、養老村、里村、池尻村、西山村、小畑村、一本松新村、神木村、上原村、中山新村、磐村
    • 上荘村 ← 見土呂村、白沢新村、井ノ口村、国包村、都染村、薬栗村、小野村
    • 東志方村 ← 細工所村、高畑村、広尾村、岡村、野尻新田村、大沢村、行常村、畑村、雑郷新村、東飯坂村、東中村、大宗村
    • 志方村 ← 志方町、西中村、西飯坂村、投松村、上富木村
    • 西志方村 ← 下原村、永室村、西牧村、山中新村、成井村、西山新村、横大路村
    • 西神吉村 ← 鼎村、宮前村、中西村、西村、大国村、岸村、辻村
    • 阿弥陀村 ← 阿弥陀村、魚橋村、生石村、南池村、北池村、北山新村、長尾新村、地徳新村（現・高砂市）
    • 別所村 ← 別所村、小林村、北宿村、佐土新村、佐土村（現・姫路市）
    • 的形村 ← 的形村、福泊村（現・姫路市）
    • 北浜村 ← 北脇村、西浜村、牛谷村（現・高砂市）
    • 大塩村（単独村制。現・姫路市）
    • 加古川町が加古郡加古川町、木村・西河原村・友沢村・稲屋村が加古郡鳩里村、唐端新村・志吹村が飾東郡谷外村、北野新村・大釜村・清住新村が飾東郡谷内村のそれぞれ一部となる。
  • 明治29年（1896年）7月1日 - 郡制を施行。
  • 大正2年（1913年）4月1日 - 曽根村が町制施行して曽根町となる。（1町14村）
  • 大正12年（1923年）4月1日 - 郡会が廃止。郡役所は存続。
  • 大正15年（1926年）
    • 4月1日 - 大塩村が町制施行して大塩町となる。（2町13村）
    • 7月1日 - 郡役所が廃止。以降は地域区分名称となる。
  • 昭和3年（1928年）11月10日 - 米田村が町制施行して米田町となる。（3町12村）
  • 昭和29年（1954年）
    • 7月1日 - 曽根町・伊保村が加古郡高砂町・荒井村と合併して高砂市が発足し、郡より離脱。（2町11村）
    • 8月1日 - 志方村・東志方村・西志方村が合併して志方町が発足。（3町8村）
  • 昭和30年（1955年）4月1日 - 上荘村・平荘村が加古川市に編入。（3町6村）
  • 昭和31年（1956年）9月30日（2町3村）
    • 東神吉村・西神吉村および米田町の一部（平津・船頭）が加古川市に編入。
    • 阿弥陀村および米田町の残部（米田・米田新・古新・塩市・島・神爪）が高砂市に編入。
  • 昭和32年（1957年）
    • 3月10日 - 北浜村が高砂市に編入。（2町2村）
    • 10月1日 - 別所村が姫路市に編入。（2町1村）
  • 昭和33年（1958年）1月1日 - 的形村が姫路市に編入。（2町）
  • 昭和34年（1959年）5月1日 - 大塩町が姫路市に編入。（1町）
  • 昭和54年（1979年）2月1日 - 志方町が加古川市に編入。同日印南郡消滅。
