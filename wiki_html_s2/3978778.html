洲羽国造（すわのくにのみやつこ、すわこくぞう）は、のちに信濃国となる地域（科野国南部）を支配した国造である。

概要[編集]

表記[編集]

『修補諏訪氏系図.正編』[1][注釈 1]、『諸系譜』[2]に洲羽国造、『先代旧事本紀』「国造本紀」の刊本[注釈 2][注釈 3]に須羽国造との記載があり、それ以外の史料には洲羽国造は見られない。

祖先[編集]

  • 『修補諏訪氏系図.正編』によると建御名方神の後裔（「建御名方神 ー 伊豆早雄命 ー 箭津安賀多命 ー 健志名乃命 ー 武美名別命 ー 武美名別彦命 ー 八重隈根命 ー 弟足男命 ー 加禰足彦命 ー 武沼河命 ー会知早雄命 ー 真曽我男命 ー 建大臣命」）の建大臣命が景行天皇朝に初代洲羽国造に任命されたと伝わる。
  • 『諸系譜』第六冊、第十三冊、第二十八冊に掲載された各種「神氏」・「諏訪氏」の系図によると、建御名方神の後裔の兄武彦命が稚桜宮朝（神功皇后摂政期）に初代洲羽国造に任命されたと伝わる。ただし兄武彦命は武国彦命の子に置かれているため、兄武彦命の実名が健国津見命であると見られる[4]。また第十三冊では神功皇后摂政期任命の注記を消してある[5]。
  • 各史料によって建大臣命（諸系譜第六冊では武国彦命、第十三冊では両方を記載）は建御名方神の御子神出早雄命の14世、12世孫、11世孫、6世孫と伝わる。
  • 『古代氏族系譜集成』上巻記載の『諏訪氏系図』では、その他系図との比較検討や注記の内容から「建御名方神―（4代略）―会知速男命（市速男命）―真曽我男命―武国彦命（建大臣命）」と整理されている[4]。

氏族[編集]

洲羽氏（すわうじ、姓は君、神氏とも）。三輪氏と同系の海神族で龍蛇信仰を持つ[6]。また『天孫本紀』には須羽直（すわのあたい）が登場し、直を称していたともされる。乙穎の2世孫・国積の時代に庚午年籍で神人部氏（みわひとべうじ）で直を、後に宝亀三年に宿禰を賜り、再び諏訪氏を名乗る。なお乙穎以降の神人部氏は洲羽君の跡に科野国造一族の金刺舎人直から嗣が入ったもので、その後に神人部直姓に改めたものとされる[7]。

神人部直氏は三輪氏後裔からも輩出され、埴科郡に居住した神人部子忍男はこの一族出身である。

入諏時期については諸説あるが、神氏の諏訪入りは神武東征によって磯城県主・三輪氏系の建御名方神や伊豆速雄命が大和から敗走した弥生時代の頃とする説がある[8]。実際に記紀から想定される建御名方神の逃走経路上に、龍蛇信仰を持つ海神族の祭器であった銅鐸が多数分布し、塩尻市からも三遠式銅鐸が出土している。また長野市三輪の地に式内美和神社を祀り、諏訪大社上社北方の大和（大輪）が諏訪郡美和郷と呼ばれ、上伊那郡辰野町にも三輪神社が鎮座し、更に諏訪大社に鉄鐸が伝わっていることに注目される[9]。

本拠[編集]

国造の本拠は不明。のちの信濃国諏訪郡とも。諏訪には代々諏訪氏が奉斎した諏訪大社が鎮座し、周辺には5世紀前期のフネ古墳、5世紀中期の片山古墳が存在する。ただし科野国の古墳造営は、4世紀前期から始まる埴科古墳群など千曲川流域の北信（科野国造の支配領域）と、5世紀後半から始まる飯田古墳群など天竜川流域の南信（高森町以南）が中心であるため、諏訪に国造が置かれたとするのは疑問ともされる[10]。ただし飯田古墳群の築造勢力については定説がなく、南信には阿智村を本拠とする阿智祝部が存在した。また南信に諏訪氏の奉斎社と思われる古社も存在しない。

支配領域[編集]

洲羽国造の支配領域は当時科野国と呼ばれていた地域の南部（諏訪郡、伊那郡）とされる。律令制の後、信濃国から一時諏訪国が分立したことがあり、これが旧洲羽国の領域ともされる[11]。

『古事記』には、大国主神の子建御名方神が諏訪に入国する際に、「科野国の洲羽海」に至ると記される。

神代、出雲の国譲りに反対していた大国主神の次男・建御名方神が天津神の建御雷神と争ってとって敗れ、助命されて科野国洲羽に住まわされたと伝わる。その地には諏訪大社が建立され、信濃国一の宮の格式を誇り武芸と開拓の神として尊崇を集めた。

弥生時代から古墳時代にかけての科野は、更級・埴科を中心とした千曲川流域であり、県内最大の前方後円墳で科野の大王の墳墓と目されている森将軍塚古墳を筆頭とした埴科古墳群が残されている現在の千曲市（旧更埴市）から川柳将軍塚古墳のある長野市南部（旧更級郡）にかけての一帯が科野の中心（科野国造）であったとされる[12]。一方諏訪地域に前方後円墳が造営されるのは下諏訪町の下諏訪青塚古墳が唯一であり、この古墳は下社奉斎氏族の大祝金刺氏（多氏同系）の奥津城と考えられる[注釈 4]。

洲羽国は7世紀には既に信濃国の諏訪郡となっていたが、どの時期に統合されたのかは不明である。

氏神[編集]

奉斎社[編集]

旧諏訪郡にあり信濃国一宮の諏訪大社（すわたいしゃ、位置）か。上社の大祝は洲羽国造の後裔諏訪氏がつとめたが、神長官の守矢氏が大きな実権を握った。

関連神社[編集]

  • 洩矢神社（もりやじんじゃ）
    長野県岡谷市川岸にある神社。祭神の洩矢神は諏訪氏祖の建御名方神が入諏する以前に諏訪（守屋山山麓）に鎮座した神とされ、その後裔たる守矢氏が諏訪上社の神長官を務めた。ただし『神長官系譜』には建御名方神の子・片倉辺命の後裔とされ[13]、別伝では洲羽国造後裔の畔野君後裔である御枝の孫・山麻侶に「神長官祖」という譜註が見える[14][15]。
  • 出早雄小萩神社（いずはやおこはぎじんじゃ）
    長野県岡谷市に鎮座する神社。もともと二つだった神社を合祀したもので、国造の祖である伊豆早雄命と弟の興波岐命を祀る。
  • 会津比売神社（かいづひめじんじゃ）
    長野県長野市に鎮座する神社。祭神の会津比売神は伊豆早雄命の娘（会知早雄命の娘の転化か）で、初代科野国造の武五百建命の妻とされる。
  • 会地早雄神社（おうじはやおじんじゃ）
    埴科郡坂城町に鎮座する神社。現在は熊野神と大穴持命を祀るが、名称から本来は会知早雄命を祀ったものか。
  • 健御名方富命彦神別神社（たけみなかたとみのみことひこかみわけじんじゃ）
    建御名方神の御子・建御名方彦神別命を祀り、彦神別命は洲羽国造の祖とも伝わる。
  • 武水別神社（たけみずわけじんじゃ）
    長野県千曲市に鎮座する神社。祭神の武水別大神は建御名方神後裔と伝え、洲羽国造の祖神の一柱に数えられる。
  • 可毛羽神社（かもはじんじゃ）
    長野県長野市に鎮座する神社。現在は諏訪神夫婦を祀るが、名称から本来は伊豆早雄命の御子・可毛羽神を祀ったものか。
  • 五加八幡神社（ごかはちまんじんじゃ）
    長野県上田市に鎮座する神社。国造の祖である八縣宿祢命を祀る。

墓[編集]

洲羽国造歴代の奥津城は不明であるが、フネ古墳出土の蛇行剣は龍蛇信仰との関わりも指摘され[16][17][18]、国造祖神の建御名方神を蛇（あるいは龍）とみなす信仰は古くから伝わる[19][20][21]事情もある。

人物[編集]

『修補諏訪氏系図.正編』及び『古代氏族系譜集成』に記される洲羽国造を以下に記載する。ただし実際には健隈照命は建大臣命の孫と見られるため[22]、同人異名が一代分重複されている可能性がある。

  • 建大臣命（たけおおおみ の みこと、武国彦命/たけくにひこ の みこと）
    初代洲羽国造。景行朝の人物で建御名方神の7世孫。
  • 兄武彦命（えたけひこ の みこと）
    建大臣命の子。稚桜宮朝の国造で、弟の弟武彦命は同朝に初代木蘇国造に任命されたと伝わる。
  • 健国津見命（たけくにつみ の みこと、大隈富命/おおくまとみ の みこと）
    兄武彦命の子とされるが、前後の世代関係から兄武彦命の実名か[23]。
  • 健隈照命（たけくまてる の みこと、武水上命/たけみなかみ の みこと）
    科野国造健甕富命の娘を娶る。武諸日命が幼少にして健甕富命が死んだため、科野国造を兼任したとされる。八重隈根命と同人。
  • 武水隈命（たけみぐま の みこと）
    健隈照命の子。
  • 武背男命（たけせお の みこと）
    健隈照命の子とも武水隈命の子とも伝わる。
  • 武河隈君（たけかわくま の きみ）
    武背男命の子。国造表記はないが須羽君姓を賜る。科野国造後裔の伊努古君と同代であるため、允恭天皇朝の人物か。

女性[編集]

  • 妹古（いもこ）
    須羽直（すわのあたい）の娘とされ、物部麻佐良の妻として物部氏に嫁ぎ、物部荒甲を生む。
  • 乙比売（おとひめ、弟兄部/おとえべ）
    倉見君の娘とされ、金刺麻背の妻として科野国造に嫁ぎ、初代諏訪大祝の乙穎と金刺倉足を生む。

子孫[編集]

  • 神人部宿祢（みわひとべのすくね）
    武河隈君の弟・山守君の後裔で、庚午年籍に神人部宿祢を賜姓された。
  • 守矢祝（もりやのほおり）
    神人部宿祢の後裔で、諏訪大社神長官を歴任する。
