『新すばらしきこのせかい』（英語: NEO:The World Ends with You）は、スクウェア・エニックスより発売されたゲームソフト。Nintendo Switch/PlayStation 4版は2021年7月27日に発売。PC（Epic Games Store）版は2021年夏発売予定。

2007年に発売されたニンテンドーDS用ソフト『すばらしきこのせかい』の14年ぶりの続編。設定や世界観は2021年4月 - 6月まで放送されたテレビアニメ『すばらしきこのせかい The Animation』に準拠し、同アニメ版の最終回から3年後の物語が描かれる[1]。

ツイスターズ[編集]

リンドウ / 奏 竜胆（かなで りんどう）
声 - 内田雄馬
本作品の主人公。黒いマスクを付けた男子高校生。バッジを手に入れたことでフレットとともに死神のゲームに参加することになる。
よくスマホをいじっており、ネットに「スワロウさん」という友達がいる。
周囲の空気を読んで行動する性格で、あまり社交的ではない。フレットに指名されチーム『ツイスターズ』のリーダーとなる。
固有サイキックは「リスタート」。同日の過去の時間に戻ることができる能力。
フレット / 觸澤 桃斎（ふれさわ とうさい）
声 - 沢城千春
リンドウの友人で、彼ともに死神のゲームに参加する。
お調子者で、誰に対してもタメ口で話す。
人と楽しく会話することが好きで、誰とでも打ち解けられる特技を持つ。一方で重苦しい空気を嫌い、真面目な話し合いを避けている。
固有サイキックは「リマインド」。イメージを相手の脳内に送り、忘れていることを思い出させるできる能力。
ナギ / 笛吹 梛（うすい なぎ）
声 - 片平美那
死神のゲームの途中で出会ったオタク気質の女子大生。ツーサイドアップが特徴的で、「推しが尊い」と書かれた白いTシャツを着ている。
洞察力があり、素の自分を出さない人間に対して冷たい。
固有サイキックは「ダイブ」。他者の心に飛び込む能力で、それを応用し対象に憑りついているノイズを排除し、正気に戻すことができる。
ミナミモト / 南師 猩（みなみもと しょう）
声 - 藤本隆行
死神のゲームの参加者。
黒ずくめの服装が特徴的な褐色の青年で、数学用語を使用して会話をする。
死神のゲーム初日で強引にツイスターズに入るが、一人で行動することが多い。戦い慣れしており、死神のゲームに詳しい。

ルーイン[編集]

ススキチ / 周々木 鹿一（すすき かいち）
声 - 花江夏樹
死神のゲームの参加者。
巨大な身体と鋭い眼光が特徴的で、声が高く、まくし立てるように喋る。スマホで文を打つのが下手。
ゲーム、特にリバーシが大好きで、白黒ハッキリしている性格。
ツグミ / 松苗 亜未（まつなえ つぐみ）
声 - 尾崎由香
死神のゲームの参加者。
黄土色の長髪が特徴的な少女で、持ち歩いているぬいぐるみは、現在渋谷で話題のブランド『ガット・ネーロ』の黒猫キャラクターに酷似している。

幽川舎[編集]

フウヤ / 河原 封也（かわはら ふうや）
声 - 中島ヨシキ
死神のゲームの参加者で、幽川舎のリーダー。臆病な性格で、暗渠が好き。

ヴァリー[編集]

カノン / 立花 果遠（たちばな かのん）
声 - 上坂すみれ
死神のゲームの参加者で、ヴァリーのリーダー。スタイル抜群の美女。
普段の言動は優しいが、狡猾な面も持ち合わせている。

ピュアハート[編集]

モトイ / 穴沢 基威（あなざわ もとい）
声 - 駒田航
死神のゲームの参加者で、ピュアハートのリーダー。当たり前なことを詩っぽく言う。普段はビジネス用語を多用して会話している。

渋谷死神[編集]

カリヤ / 狩谷 拘輝（かりや こうき）
声 - 勝杏里
落ち着いた言動が特徴的な、渋谷の男性死神。常に飴玉を持ち歩いている。
出世に興味がなく下っ端の死神として仕事を続けている。
ヤシロ / 八代 卯月（やしろ うづき）
声 - 新井里美
昔、カリヤと行動を共にしていた気の強い渋谷の女性死神。ピンクのショートヘアとビジネススーツが特徴的。
ココ / 新 子々（あたらし ここ）
声 - 高野麻里佳
死神の少女。気まぐれでずる賢い。

新宿死神[編集]

シイバ / 三谷風 椎葉（みやかぜ しいば）
声 - 濱野大輝
渋谷にやってきた新宿死神のリーダーで、死神のゲームのゲームマスター。金髪をオールバックにしている。
常に熱いゲーム展開を心から望んでいる。
クボウ / 久網 旦蔵（くぼう たんぞう）
声 - 中野泰佑
中年男性のような姿の死神。嫌味ったらしい言動が多い。「ンハッ」という独特な笑い方をする。
アヤノ / 蒲池 菖乃（かまち あやの）
声 - 佐倉綾音
肩に水色のイグアナを乗せている女死神。いわゆる死神の幹部であり、能力も高くリーダーのシイバから信頼されている。
ショウカにだけは優しい。
カイエ / 小野 解依（おの かいえ）
声 - 千葉翔也
占いをやっている死神で、無口無表情。
スマホで会話する時はよく喋る上に面倒見が良い。
ヒシマ / 坂筑 菱真（さかずき ひしま）
声 - 武内駿輔
眼鏡をかけた知的な雰囲気の死神。ゲームの運営にあまり参加していない。ひねくれ者かつ毒舌家。
ショウカ / 桜音 紫陽花（さくらね しょうか）
声 - 鬼頭明里
気だるげな雰囲気の女死神。
『ガット・ネーロ』のパーカーを着ている。ツイスターズの前に現れて煽り、「バイ」と言い残し去っていくことが多い。
