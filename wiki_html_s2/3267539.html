算術の超準モデル

算術の超準モデル (英: non-standard model of arithmetic) とは、（一階）ペアノ算術のモデルのうち、通常の自然数ではない元（超準数）を含むようなモデルのことである。それに対し、通常の自然数 <math>\mathbb{N}</math> は算術の標準モデルと呼ばれる。ペアノ算術の任意のモデルは線形順序で並んでおり、<math>\mathbb{N}</math> と同型な切片を持つ。超準モデルは、その切片の外に元を持つようなモデルであると言える。

存在[編集]

算術の超準モデルの存在を証明する方法はいくつか存在する。

コンパクト性定理による方法[編集]

コンパクト性定理を用いて超準モデルの存在を示すことができる。証明の概略は、<math>c</math> を新たな定数として、ペアノの公理系 <math>\mathrm{PA}</math> に <math>\{n < c : n = 1, 2, 3,...\}</math> という形の無限個の公理を付け加えた公理系 <math>\mathrm{PA}^{\ast}</math> を考え、コンパクト性定理により <math>\mathrm{PA}^{\ast}</math> を満たすモデル <math>\mathbb{N}^{\ast}</math> の存在を示すというものである[1]。<math>\mathrm{PA}^{\ast}</math> はペアノの公理系を拡張したものであるため、当然ペアノの公理を満たしている。また通常の自然数では定数 <math>c</math> をいかように解釈しても <math>\mathrm{PA}^{\ast}</math> を満たすようにはできないため、<math>c</math> は超準数であり、<math>\mathbb{N}^{\ast}</math> は超準モデルとなる。

<math>\mathrm{PA}^{\ast}</math> にコンパクト性定理を適用するには、その任意の有限部分 <math>T</math> がモデルを持つことを示せばよい。<math>T</math> は <math>\mathrm{PA}</math> の部分集合に <math>n_1 < c, n_2 < c, \ldots, n_m < c</math>（<math>n_1 < n_2 < \cdots < n_m</math>）という有限個の公理を付け加えた形をしているため、<math>c</math> の解釈を <math>n_{m}+1</math> と定めれば、自然数 <math>\mathbb{N}</math> が <math>T</math> のモデルになっていることが言える。

ゲーデルの不完全性定理による方法[編集]

不完全性定理により、標準モデルでは真であるがペアノの公理系においては決定不能であるような文（ゲーデル文）<math>G</math> が存在する。このとき、完全性定理より、ペアノの公理系 <math>\mathrm{PA}</math> に <math>\lnot G</math> を加えた公理系にモデルが存在する。標準モデルで <math>G</math> は真なので、このモデルは超準モデルでなければいけない。このように、<math>\lnot G</math> を満たすことは、そのモデルが超準的である為の十分条件となる。しかし、これは必要条件ではない。いかなるゲーデル文 <math>G</math> に対しても、<math>G</math> が真であるようなあらゆる濃度のモデルが存在する。

算術が無矛盾であると仮定すれば、算術に <math>\lnot G</math> を付け加えたものもまた無矛盾である。しかし、<math>\lnot G</math> は算術が矛盾していることを意味するのだから、結果得られた算術の体系は ω-無矛盾にはならない（なぜなら、<math>\lnot G</math> は偽であり、したがって ω-無矛盾性に反する）。

超積による方法[編集]

算術の超準モデルを構成するもうひとつの方法は超積に基づくものである。典型的な構成では自然数列全体の成す集合 <math> \mathbb{N}^{\mathbb{N}} </math> を用いる。2つの列が同一視されるのは、それらがある固定された非単項超フィルターに属す添字集合の上で一致するときである。このようにして得られた半環は算術の超準モデルとなる。これは超自然数と同一視出来る。

可算超準モデルの構造[編集]

超積モデルは非可算となることが知られている。このことを見るには <math>\mathbb{N}</math> の無限直積から超積モデルへの単射を構成すればよい。他方でレーヴェンハイム-スコーレムの定理により、算術の可算な超準モデルが存在しなければならない。構成法の一つとしてヘンキン構成を用いた方法がある。

算術の超準モデルの順序構造は、ある端点を持たない稠密全順序集合 <math>\mathcal{Q}</math> を用いて、<math>\mathbb{N} \oplus \mathbb{Z} \times \mathcal{Q}</math> と表せることが知られている（後述）。可算超準モデルの場合、上の表示において、<math>\mathcal{Q}</math> は可算となるので、端点を持たない稠密全順序の理論（DLO）の可算範疇性より、<math>\mathcal{Q}</math> は <math>\mathbb{Q}</math> と同型となる。したがって可算超準モデルの順序構造は <math>\mathbb{N} \oplus \mathbb{Z} \times \mathbb{Q}</math> と表せる。

いま <math>M</math> を算術の超準モデルとする。<math>M</math> は <math>\mathbb{N}</math> を含んでいるものと見做せる。ペアノ算術では、各自然数 <math>n</math> に対して、

<math>\forall x \left( \bigwedge_{k=0}^{n} (x \neq n) \to n < x \right)</math>

が証明可能である。したがって <math>M</math> に於いて真である。このことから <math>M</math> は <math>\mathbb N</math> の後ろに無限大元からなる部分を繋げたような順序構造をしていることが分かる。すなわち、順序集合として<math> M = \mathbb{N} \oplus (M\setminus \mathbb{N})</math> が成り立つ。

次に <math>M' = M \setminus \mathbb{N}</math> の順序構造を調べる。いま <math>M'</math> 上の二項関係

<math>x E y \iff x-y \in \mathbb{Z}</math>（つまりある自然数 <math>n</math> に対して <math>x=y+n</math> または <math>y=x+n</math>）

を考える。これは同値関係となる。各々の同値類は

<math>\left[x\right] = \{x + n | n\in \mathbb{Z}\}</math>

の形をしていることが分かる。商集合 <math>\mathcal{Q}=M'/E</math> 上の二項関係を

<math>\left[x\right] \ll \left[y\right] \iff x < y \text{ and } y-x \in M' </math>

で定めると、これはwell-definedであって、全順序となることが分かる。このとき、<math>M'</math> の2元 <math>u,v</math> について、<math>u < v</math> であることと、次の条件は同値である：

  • <math>\left[u\right] = \left[v\right]</math> であって、<math>u = x+n , v=x+m</math> と書いたとき <math>n<m</math>；もしくは
  • <math>\left[u\right] \ll \left[v\right]</math>

したがって <math>M'</math> の順序は <math>\mathbb{Z} \times \mathcal{Q}</math> 上の逆辞書式順序となっている。

最後に <math>\mathcal{Q}</math> の順序構造を調べる。任意の <math>x\in M'</math> に対して <math>\left[x\right] \ll \left[2x\right]</math> が成り立つ。任意の元は偶数または奇数であることはペアノ算術で証明できる。よって <math>x</math> はある <math>y \in M</math> によって <math>x=2y,2y+1</math> のいずれかの形に表せる。またこのとき <math>y \in M'</math> であり、<math>\left[y\right] \ll \left[ x\right]</math> が成り立つ。したがって <math>\mathcal{Q}</math> は端点（最大元と最小元）を持たない。次に <math>x,y \in M'</math> について <math>\left[x\right] \ll \left[y\right]</math> と仮定する。<math>x+y</math> は偶数または奇数だから、ある <math>z \in M</math> によって <math>x+y=2z,2z+1</math> のいずれかの形に表せる。いずれにしても <math>z\in M'</math> であり、<math>\left[x\right] \ll \left[z\right] \ll \left[ y\right]</math> が成り立つ。すなわち <math>\mathcal{Q}</math> は稠密である。

以上より、<math>M = \mathbb{N} \oplus \mathbb{Z} \times \mathcal{Q}</math> であり、<math>\mathcal{Q}</math> は端点を持たない稠密全順序集合である。
