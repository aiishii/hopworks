    「YOUR NAME NEVER GONE / Now or Never / You Got Me」（ユアー・ネーム・ネヴァー・ゴーン / ナウ・オア・ネヴァー / ユー・ガット・ミー）は、2003年11月19日に発売されたCHEMISTRYの通算9枚目のシングル。

    解説[編集]

    CHEMISTRY2枚目となる、トリプルA面シングル。また、トリプルタイアップもついた。

    トリプルA面扱いであるが、メディアでは「YOUR NAME NEVER GONE」のみが扱われ、前面ジャケットにはYOUR NAME NEVER GONEとしか表記されていない。

    「YOUR NAME NEVER GONE」と「Now or Never」は、後発のアルバム『One×One』に収録。「You Got Me」のみ、未収録となった。(後のベスト・アルバム『ALL THE BEST』には3曲とも収録）。

    収録曲[編集]

    1. YOUR NAME NEVER GONE
      作詞：麻生哲朗／作曲：SPANOVA／編曲：森俊之
      ソニー『ウォークマン』/『サウンドゲート』CMソング。CMには柴咲コウとオダギリジョーが主演。
    2. Now or Never / CHEMISTRY meets m-flo
      作詞：m-flo、CHEMISTRY／作曲・編曲：m-flo
      フジテレビ系列・テレビアニメ『アストロボーイ・鉄腕アトム』オープニングテーマ曲。
      m-floとのコラボレート曲で、m-flo loves CHEMISTRYの「Astrosexy」がこの曲の別バージョンにあたる。ミュージック・ビデオでも『アストロボーイ・鉄腕アトム』とのコラボをしており、全編アニメの映像が使用されている。曲のサビ部分ではアニメの登場人物が歌に合わせてリップシンクする映像が新規に制作されている[1]。
    3. You Got Me
      作詞：角田誠／作曲：DAICHI／編曲：森俊之
      ダイハツ『ムーヴカスタム』CMソング。
      スガシカオがギターで参加。
    4. YOUR NAME NEVER GONE (Instrumental)
    5. Now or Never (Instrumental)
    6. You Got Me (Instrumental)
