『STEREO WORXXX』（ステレオ・ワークス）は、2012年3月7日にヤマハミュージックコミュニケーションズより発売されたcapsuleの13枚目（ベストアルバムを含むと14枚目）のオリジナルアルバムである[2]。

背景について[編集]

2011年12月27日にニュー・アルバムが翌年3月に発売されることが発表され[3]、前作『WORLD OF FANTASY』より約9ヶ月ぶりのアルバムとなった[4]。

capsuleは本作をリリースした翌年にレーベルを移籍したことで[5]、本作はcapsuleがヤマハミュージックコミュニケーションズ及びcontemodeレーベルからリリースした最後のオリジナルアルバムとなった[6]。

制作について[編集]

「Step on The Floor」のレコーディングの際、こしじまは「『Step』の発音がうまくいかなくて、何度も録り直しているうちにゲシュタルト崩壊を起こし」たことから、本作の難点として挙げている[7]。

ミックスの方向性をアルバム全体で統一しなくてもいいと中田が思い、ドラムのキックの位置やベースのバランスが曲ごとに違う[7]。

前作とは違うギターとマイクが使用された[7]。

ミュージック・ビデオについて[編集]

Feelin' Alright[編集]

フォトコミュニケーションサービス「cotto」とのコラボレーション企画で制作され[8]、2012年4月2日にYouTubeで公開された[9]。

楽曲について[編集]

6.Tapping Beats

『LIAR GAME -再生- オリジナル・サウンドトラック』に別バージョンが収録されている[7]。

9.Transparent

普通のメロディーラインだが、一般的なコード進行を全部外していくから次の展開が予想できない楽曲[10]。それゆえ、「歌があることの大事さがわかる。」と中田は語っている[10]。

タイアップについて[編集]

  • 映画『LIAR GAME -再生-』主題歌（#5[11]）
  • 映画『LIAR GAME -再生-』挿入歌（#7[11]）
  • アプリ「チャリ走DX」capsule stage BGM（#8[12]）

パッケージについて[編集]

  • 初回盤：2CD
  • 通常盤：CD

前作同様、初回盤（品番：YCCC-10023/B[13]）と通常盤（品番：YCCC-10024[2]）の2形態で発売された。初回盤には本作収録曲のリミックス・バージョンが収録されたボーナスディスクが付く[13]。

プロモーションについて[編集]

同年2月1日より「Step on The Floor」が先行配信された[14]。

『capsule "STEREO WORXXX" RELEASE PARTY』について[編集]

capsule "STEREO WORXXX" RELEASE PARTY
capsule の クラブ・ツアー
関連アルバム 『STEREO WORXXX』
初日 2012年4月7日
最終日 2012年7月16日
公演数 6[15]

『capsule "STEREO WORXXX" RELEASE PARTY』は、capsuleが2012年春から夏にかけて開催した全国クラブツアー[15]。

映像や音源の販売は行われていない。

日程について[編集]

当初福岡公演は同年5月3日に予定されていた[15]が、会場の休業により延期され[16]、会場を変更して開催された[17]。
