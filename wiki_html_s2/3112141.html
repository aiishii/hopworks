加群の層

数学において，O 加群の層 (sheaf of O-modules) あるいは単に環付き空間 (X, O) 上の O 加群 (O-module) とは，層 F であって，X の任意の開部分集合 U に対し，F(U) が O(U) 加群であり，制限写像 F(U) → F(V) が制限写像 O(U) → O(V) と整合的なもの，すなわち O(U) の任意の f と F(U) の任意の s に対し，fs の制限が f の制限と s の制限との積であるものである．

標準的な場合は X がスキームで O がその構造層であるときである．O が定数層（英語版） <math>\underline{\mathbf{Z}}</math> のとき，O 加群の層はアーベル群の層（すなわちアーベル層）と同じである．

X が環 R の素スペクトルであるとき，任意の R 加群は自然に OX 加群を定義する（associated sheaf と呼ばれる）．同様に，R が次数環で X が R の Proj（英語版） であるとき，任意の次数加群は自然に OX 加群を定める．そのように生じる O 加群は準連接層の例であり，実は，アファインあるいは射影スキーム上，すべての準連接層はこのようにして得られる．

環付き空間上の加群の層はアーベル圏をなす[1]．さらに，この圏は充分単射的対象を持ち[2]，したがって層係数コホモロジー <math>\operatorname{H}^i(X, -)</math> を大域切断関手（英語版） <math>\Gamma(X, -)</math> の i 次右導来関手として定義でき，実際そう定義する[3]．

例[編集]

  • 環付き空間 (X, O) が与えられ，F が O の O 部分加群であるとき，それは O のイデアルの層あるいはイデアル層（英語版）と呼ばれる，なぜならば X のすべての開部分集合 U に対し，F(U) は環 O(U) のイデアルであるからである．
  • X を n 次元の滑らかな代数多様体とすると，X の接層（英語版）は余接層（英語版） <math>\Omega_X</math> の双対であり，標準層 <math>\omega_X</math> は <math>\Omega_X</math> の n 次外冪（行列式）である．

演算[編集]

(X, O) を環付き空間とする．F と G が O 加群のとき，それらのテンソル積は，

<math>F \otimes_O G</math> あるいは <math>F \otimes G</math>

と表記され，前層 <math>U \mapsto F(U) \otimes_{O(U)} G(U)</math> に伴う層である O 加群である．（層化が避けられないことを見るには，<math>O(1) \otimes O(-1) = O</math> の大域切断を計算せよ，ここで O(1) は射影空間上のセールの捩り層（英語版）である．）

同様に，F と G が O 加群のとき，

<math>\mathcal{H}om_O(F, G)</math>

は層 <math>U \mapsto \operatorname{Hom}_{O|_U}(F|_U, G|_U)</math> である O 加群を表す[4]．とくに，O 加群

<math>\mathcal{H}om_O(F, O)</math>

は F の双対加群と呼ばれ，<math>\check F</math> と表記される．注意：任意の O 加群 E, F に対し，自然な準同型

<math>\check{E} \otimes F \to \mathcal{H}om_O(E, F)</math>

が存在し，E が階数有限の局所自由層であるとき，これは同型である．とくに，L が階数 1 の局所自由であるとき（そのような L は可逆層あるいは直線束と呼ばれる）[5]，これは

<math>\check{L} \otimes L \simeq O</math>

となり，したがって可逆層の同型類たちは群をなす．この群は X のピカール群と呼ばれ，一次コホモロジー群 <math>\operatorname{H}^1(X, \mathcal{O}^*)</math> と（チェックコホモロジー（英語版）による標準的な議論によって）カノニカルに同一視される．

E が階数有限の局所自由層であれば，ペアリングによって与えられる O 線型写像 <math>\check{E} \otimes E \simeq \operatorname{End}_O(E) \to O</math> がある；それは E の跡写像（英語版）と呼ばれる．

任意の O 加群 F に対し，F のテンソル代数，外積代数，対称代数が同じように定義される．例えば，k 次外冪

<math>\wedge^k F</math>

は前層 <math>U \mapsto \wedge^k_{O(U)} F(U)</math> に伴う層である．F が階数 n で局所自由ならば，<math>\wedge^n F</math> は F の行列式直線束（英語版）と呼ばれ（技術的には可逆層であるが），det(F) と書かれる．自然な完全対

<math>\wedge^r F \otimes \wedge^{n-r} F \to \operatorname{det}(F)</math>

がある．

f: (X, O) →(X', O') を環付き空間の射とする．F が O 加群のとき，順像層（英語版） <math>f_* F</math> は自然な写像 O' →f*O により O' 加群である（そのような自然な写像は環付き空間の射のデータの一部である）．

G が O' 加群であるとき，G の逆像 <math>f^* G</math> である加群は加群のテンソル積として与えられる O 加群である：

<math>f^{-1} G \otimes_{f^{-1} O'} O</math>

ただし f−1G は G の逆像層（英語版）であり，<math>f^{-1} O' \to O</math> は <math>O' \to f_* O</math> から随伴によって得られる．

<math>f_*</math> と <math>f^*</math> の間には随伴の関係がある：任意の O 加群 F と O′ 加群 G に対して，アーベル群として

<math>\operatorname{Hom}_{O}(f^* G, F) \simeq \operatorname{Hom}_{O'}(G, f_*F)</math>

が成り立つ．projection formula（英語版）もある： O 加群 F と階数有限の局所自由 O′ 加群 E に対して，

<math>f_*(F \otimes f^*E) \simeq f_* F \otimes E</math>

が成り立つ．

性質[編集]

(X, O) を環付き空間とする．O 加群 F が大域切断によって生成される (generated by global sections) とは，O 加群の全射

<math>\oplus_{i \in I} O \to F \to 0</math>

が存在することをいう．明示的にはこれは，F の大域切断 si であって，各茎 Fx における si の像たちが Fx を Ox 加群として生成するものが存在することを意味する．

そのような層の例は代数幾何学において任意の可換環のスペクトル Spec(R) 上に R 加群 M を付随させるものである．別の例：カルタンの定理Aにより，シュタイン多様体上の任意の連接層は大域切断によって張られる（cf. 下記のセールの定理A）．スキームの理論において，関連する概念は豊富な直線束である．（例えば，L が豊富な直線束であれば，それのある冪が大域切断によって生成される．）

移入 O 加群は脆弱である（すなわちすべての制限写像 F(U) → F(V) が全射である）[6]．脆弱層はアーベル層の圏において非輪状であるから，これは次のことを意味する：O 加群の圏における大域切断関手 <math>\Gamma(X, -)</math> の i 次右導来関手がアーベル層の圏における通常の i 次層コホモロジーと一致する[7]．

加群に付随する層[編集]

M を環 A 上の加群とする．X = Spec A とおく．任意の対 <math>D(f) \subset D(g)</math> に対して，局所化の普遍性により，<math>\rho_{g, f} = \rho_{g, h} \circ \rho_{h, f}</math> なる性質を持つ自然な写像

<math>\rho_{g, f}: M[g^{-1}] \to M[f^{-1}]</math>

が存在する．すると

<math>D(f) \mapsto M[f^{-1}]</math>

は対象が集合 D(f) で射が集合の包含である圏からアーベル群の圏への反変関手である．それは実は B-層（英語版）（すなわち貼り合わせの公理を満たす）であり，したがって M に付随する層と呼ばれる X 上の層 <math>\widetilde{M}</math> を定義することを示すことができる[8]．

最も基本的な例は X 上の構造層，すなわち <math>\mathcal{O}_X = \widetilde{A}</math> である．さらに，<math>\widetilde{M}</math> は <math>\mathcal{O}_X = \widetilde{A}</math> の構造を持ち，したがって A 上の加群の圏 ModA から <math>\mathcal{O}_X</math> 上の加群の圏への完全関手 <math>M \mapsto \widetilde{M}</math> を得る．それは ModA から X 上の準連接層の圏への同型を定義し，逆射は大域切断関手 <math>\Gamma(X, -)</math> である．X がネーター的スキームのときには，関手は有限生成 A 加群の圏から X 上の連接層の圏への同型である．

構成は以下の性質を持つ：任意の A 加群 M, N に対し，

  • <math>M[f^{-1}]^{\sim} = \widetilde{M}|_{D(f)}</math>.[9]
  • A の任意の素イデアル p に対し，Op = Ap 加群として <math>\widetilde{M}_p \simeq M_p</math>.
  • <math>(M \otimes_A N)^{\sim} \simeq \widetilde{M} \otimes_{\widetilde{A}} \widetilde{N}</math>.[10]
  • M が有限表示のとき，<math>\operatorname{Hom}_A(M, N)^{\sim} \simeq \mathcal{H}om_{\widetilde{A}}(\widetilde{M}, \widetilde{N})</math>.[10]
  • <math>\operatorname{Hom}_A(M, N) \simeq \Gamma(X, \mathcal{H}om_{\widetilde{A}}(\widetilde{M}, \widetilde{N}))</math>, なぜなら ModA と X 上の準連接層の圏の間の同値があるから．
  • <math>(\varinjlim M_i)^{\sim} \simeq \varinjlim \widetilde{M_i}</math>;[11] とくに，直和を取ることと ~ は可換である．

次数付き加群に付随する層[編集]

前の節の構成と同値の次数付きの類似がある．R を R0 代数（R0 は次数 0 部分）として次数 1 の元で生成される次数環とし，M を次数 R 加群とする．X を R の Proj（英語版） とする（したがって X は射影スキームである）．するとある O 加群 <math>\widetilde{M}</math> が存在して，R の次数が正の任意の斉次元 f に対して，アフィンスキーム <math>\{f \ne 0\} = \operatorname{Spec}(R[f^{-1}]_0)</math> 上の加群の層として自然な同型

<math>\widetilde{M}|_{\{f \ne 0\}} \simeq (M[f^{-1}]_0)^{\sim}</math>

がある[12]；実際，これは <math>\widetilde{M}</math> を貼り合わせによって定義している．

例：R(1) を R(1)n = Rn+1 によって与えられる次数 R 加群とする．このとき <math>O(1) = \widetilde{R(1)}</math> はセールの捩り層（英語版）と呼ばれる（自然直線束（英語版）の双対である）．

F が X 上の O 加群のとき，<math>F(n) = F \otimes O(n)</math> と書けば，標準的な準同型

<math>(\oplus_{n \ge 0} \Gamma(X, F(n)))^{\sim} \to F</math>

が存在し，これが同型であることと F が準連接であることは同値である．

セールの定理A（英語版）により，X が射影多様体で F がその上の連接層のとき，十分大きい n に対して，F(n) は有限個の大域切断によって生成される．さらに，

層の拡大[編集]

(X, O) を環付き空間とし，F, H を X 上の O 加群の層とする．H の F による拡大 (extension) とは，O 加群の短完全列

<math>0 \rightarrow F \rightarrow G \rightarrow H \rightarrow 0</math>

である．

群の拡大と同様，F と H を固定すれば，H の F による拡大の同値類全体はアーベル群をなし（cf. Baer和（英語版）），この群は Ext 群 <math>\mathrm{Ext}_O^1(H,F)</math> と同型で，単位元は自明な拡大と対応する．

H が O のとき次が成り立つ．すべての i ≥ 0 に対して

<math>\operatorname{H}^i(X, F) = \mathrm{Ext}_O^i(O,F),</math>

なぜならば両辺とも同じ関手 <math>\Gamma(X, -) = \operatorname{Hom}_O(O, -)</math> の右導来関手だからである．

注意：著者によっては（特にハーツホーン），添え字 O を書かない．

X をネーター環上の射影スキームとする．F, G を X 上の連接層とし，i を整数とする．するとある n0 が存在して

<math>\operatorname{Ext}_O^i(F, G(n)) = \Gamma(X, \mathcal{E}xt_O^i(F, G(n))), \, n \ge n_0</math>

となる[13]．
