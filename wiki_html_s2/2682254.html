春秋航空日本株式会社 （しゅんじゅうこうくうにほん[7], Spring Airlines Japan Co., Ltd.）は、日本航空、春秋航空などが出資する日本の格安航空会社である。別称は、SPRING JAPAN（スプリング・ジャパン）であり、公式サイトや機内アナウンスなどではこちらの呼称が用いられる場合もある[3]。

概要[編集]

中国の格安航空会社春秋航空などが出資し、2012年9月7日に設立された。設立時の資本金は15億円で、航空運送事業許可の取得後に、60億円まで増資した。なお、航空法の外資規制があるため、春秋航空による出資は33%までにとどめられている[8]。出資第3位の山佐は航空機リースの他、パチスロの製造・販売でも知られる。

設立母体の春秋航空とは若干サービスを異なるものとして日本にあわせており、2017年にはオリコンが実施する日本顧客満足度調査のLCC国内線部門において価格、快適性、客室乗務員の対応など、全体的に高い評価を獲得し2年連続の首位となっている[9]。

2017年上半期、日本の航空会社では欠航率が最も高くなっていたが[10][11]、のちに2020年度は運航率首位（最もよい欠航率）に輝いた[12][13]。2017年5月から10月にかけて、安全上の理由から運航乗務員の2割にあたる、10人を乗務停止としていた[14]

2017年12月より、航空機整備の一部業務をJALエンジニアリングに委託することとなった[15]。

2020年度末には債務超過にあることが明らかとなり、2021年6月に日本航空（JAL）が過半数超の株式を追加取得し、JALグループ入りすることを同年5月に発表した[16][17]。

沿革[編集]

  • 2012年（平成24年）
    • 9月7日：会社設立。
  • 2013年（平成25年）
    • 5月：アイルランドに本拠を置く航空機リース会社AWASアビエーション・キャピタルが春秋航空日本の初号機となるボーイング737-800（機体記号:JA01GR）を受領[18]。なお、初号機は受領後も2013年7月中旬までアメリカ合衆国・アリゾナ州のピナル飛行場（英語版）で保管されていたが[19]、7月21日に日本へフェリーされた。
    • 8月2日：東京/成田 - 高松間でテストフライトを実施[20]。
    • 12月17日：国土交通省から航空運送事業許可を取得[21][22]。
    • 12月26日：機体と乗務員の制服を報道公開[23]。制服はデザイン専門学校の学生から募集し、優秀作品となったものを基にして制作した[24]。記者会見では株主の構成も明らかにした[25]。
  • 2014年（平成26年）
    • 3月25日：同年5月末からと発表していた東京/成田 - 広島線・高松線・佐賀線を同年6月27日から就航すると正式発表[26][27]。また、JTBが出資すること、運賃についても合わせて発表された[28][29]。
    • 4月8日：航空券の予約・販売を開始。
    • 5月20日：混雑空港（東京/成田）への乗入れを国土交通省航空局が許可[30]。
    • 8月1日：東京/成田 - 広島線・高松線・佐賀線をそれぞれ就航開始[31][32][33][34][35]。
    • 10月26日：冬ダイヤ開始[36]。機材繰りと、空港の受け入れ態勢が整っていないために、佐賀便を1日1便に減便する。
  • 2015年（平成27年）
    • 4月8日：成田国際空港の第3ターミナルが開業したことに伴い、第2ターミナルから移転。
    • 10月25日：冬ダイヤの開始に伴い、東京/成田 - 高松線（週4往復）を運休。[37]
    • 12月22日：ビックカメラ等が出資する投資事業組合に対する増資の実施を発表[38]。
  • 2016年（平成28年）
    • 2月13日：東京/成田 - 武漢路線に就航[39][40]。
    • 2月14日：東京/成田 - 重慶路線に就航。
    • 8月20日：東京/成田 - 札幌/新千歳路線に就航[41]。
    • 9月28日：東京/成田 - 大阪/関西路線に就航[42]。
  • 2017年（平成29年）
    • 1月28日：東京/成田 - 天津路線に就航[43]。
    • 1月29日：東京/成田 - ハルビン路線に就航[43]。
  • 2018年（平成30年）
    • 3月30日 : 樫原利幸が社長に就任[44]。
  • 2019年（令和元年/平成31年）
    • 4月10日 : 新ブランド名「SPRING」を発表[45]。
    • 12月12日 : 東京/成田 - 上海浦東路線に就航[46]
  • 2020年（令和2年）
    • 2月 : COVID-19流行により日中国際路線順次運休（ハルビン線のみ隔日運航維持）、国内線も基本平日運休、週末土日運航へ移行。
    • 10月5日 : 中国民用航空局により運航実績のある東京/成田 - 天津線の再開が認められ週一（月曜）運航再開。
  • 2021年（令和3年）
    • 1月22日 : 東京/成田 - 南京路線に週一（金曜）就航。
    • 6月9日 : 代表取締役社長に米澤章（前・日本航空中国地区総代表）が就任[47]。
    • 6月 : 日本航空（JAL）が過半数超の株式を追加取得し、JALグループ入りする[16][17]。
    • 7月1日 : 国内線の全日運航を再開。

保有機材[編集]

  • ボーイング737-800：6機
    • 座席数は189席[21]。全てエコノミークラスだが、以下のように区分が分かれており[48]、座席指定料金にそれぞれ差がある[49]。
      • コンフォートシート（1列目） - 最前方に位置する席。前方が壁となるためシートピッチが最も広い。
      • フロントシート（2・3列目） - コンフォートシートに次いでシートピッチが広い座席。
      • レッグシート（16・17列目） - 非常口座席。16列目はリクライニングが不可能。
      • スタンダードシート - 上記以外の座席。なお、機体右側のみ10列目が存在しない[48]ため、同じ料金区分の座席ではあるが5列目～15列目では右側と左側でシートピッチに差が生じている。
    • 親会社の春秋航空ではエアバスA320型機を運用しているが、日本ではボーイング機を運航する航空会社が多いため、人材確保の観点から日本国籍のLCCでは唯一、ボーイング社製旅客機を採用した[50]。
    • 就航当初の予定では2014年に3機、2015年から2017年の間毎年5機ペースでリース機として順次導入予定[51]としていたが、実際には4 - 6号機の受領が2017年以降にずれこんだ[52]。

事故、トラブル[編集]

  • 2017年10月22日、IJ701便（成田→佐賀、JA03GR）が熊本空港の東北東約40kmを飛行中、機体が動揺し客室乗務員1名が腰の骨を折る重傷を負った[65]。運輸安全委員会は航空事故と認定し、調査を進めている[66]。
  • 2017年11月末に一部報道によって「5月に佐賀空港で主翼揚力発生補助翼（フラップ）の準備ができていないまま離陸しようとした事案、10月に成田/新千歳線の飛行計画書が誤っていたものの気付かずに離陸してしまった事案などがありパイロット再訓練のため最大10名が乗務停止になっていた」事が報道[67]された。
  • 2017年11月頃から、運休[68][69]、欠航[70][71]、遅延[72]、時間変更[73]が多発し、12月には運航1か月前の運航便届け出が国土交通省航空局に出されない事案[74]も発生している。2018年になってからも1月13日から運用機材の耐空証明検査に関する機材繰りによる欠航が発生し[75]、1月16日以降は国内線は成田-広島、佐賀のそれぞれ1往復合計2往復便の運航だけ残して減便運航体制となる。2018年7月20日、6月からJALと整備子会社JALエンジニアリングへ航空機の管理を含む包括的な整備業務委託を開始したことで運休時4機運用での体制を保有6機運用の運航体制を構築し、運休路線などの再開に目途がつき、8月から成田-新千歳を再開し、増便、成田-広島を復便、成田-関西は8月に4日間だけ限定運航することを発表[76]。
