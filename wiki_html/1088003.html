『恋する乙女と守護の楯』（こいするおとめとしゅごのたて、サブタイトル：The code name is "SHIELD-9"）は、AXL（アクセル）から2007年6月29日に発売された18禁恋愛アドベンチャーゲームである。略称は「恋楯」。

2016年2月26日には、続編である『恋する乙女と守護の楯〜薔薇の聖母〜』が発売された。
