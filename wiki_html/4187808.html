L ピーター・ドイチュ (L Peter Deutsch; 1946年8月7日、マサチューセッツ州 ボストン生まれ) は、アラジン・エンタープライズの創設者であり、フリーソフトウェアの PostScript と Portable Document Format インタプリタである Ghostscript の制作者である。

ドイチュの他の仕事には、約15年後に Java のジャストインタイムコンパイル技術に影響を与えた Smalltalk の実装が含まれている。

彼についてのいくつかの物語は『Hackers: Heroes of the Computer Revolution』の本に収録されている。
